Project Overview

This project is an AR application developed in Unity, using API that get live data of weather, time and temperature. The application allows users to visualize the current weather conditions in the location through augmented reality, providing an immersive experience that combines real-world data with virtual elements. By leveraging AR technology, users can see dynamic weather effects such as rain and sunshine overlaid on their surroundings, enhancing their understanding of the current weather.

Tools and Technologies used

  • Unity
  • AR
  • C#
